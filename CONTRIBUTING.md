Thanks for your interest in improving the **Short URL API** documentation and examples!

This repository contains **docs, examples, and Postman collections** for the
myapi.rest Short URL API service. The core production API runs on myapi.rest
infrastructure and is not stored in this repository.

## How to help

You can contribute by:

- Reporting bugs or confusing behaviour in the docs or examples
- Suggesting improvements or new examples
- Proposing small fixes via pull requests

## Issues

- Use the **Bug report** or **Feature request** templates when opening issues.
- Include the endpoint, sample request/response, and any relevant context.

## Pull Requests

1. Fork the repository.
2. Create a branch for your changes.
3. Keep changes focused and small where possible.
4. Update or add examples/docs as needed.
5. Open a PR and briefly describe what you changed and why.

We aim to review contributions as time allows.

## Code of Conduct

By participating, you agree to follow our [Code of Conduct](./CODE_OF_CONDUCT.md).
